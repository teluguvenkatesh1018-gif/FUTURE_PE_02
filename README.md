# Task 2 – AI UGC Ad Script Generator

## Business Chosen
MIDNIGHT LOVE Café  
Location: Hyderabad

MIDNIGHT LOVE is a late-night café designed for students, couples, friends, and night owls looking for a cozy place to relax and enjoy chai.

---

## Objective
The goal of this task was to design a prompt system that generates high-converting UGC-style ad scripts for social media platforms like Instagram Reels and Shorts.

---

## Prompt Strategy
The prompt includes:
- Target audience
- Business details
- Tone of communication
- Ad framework (hook → story → CTA)

This ensures the generated ads feel authentic and relatable like real customer experiences.

---

## Tool Used
ChatGPT

---

## Outcome
Using the prompt system, multiple hooks and UGC ad scripts were generated that can be used for Instagram ads or promotional videos for the café.
